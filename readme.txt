Mobile App API Wordpress plugin lets you turn your website into a full-featured mobile application in minutes using Mobile App Builder.
== Description ==

Mobile App API Wordpress plugin allows you to seamlessly turn your website into a full-featured mobile application in minutes using Mobile App Builder. Allow your users to view posts, comment, like, share your content, and much more from mobile devices like iPhones, iPods, iPads or Android based phones.

Mobile App Builder is a web platform that allows you to turn your website into a full-featured mobile app in minutes. Integrate content from your Wordpress site with RSS feeds, YouTube videos, podcasts to have all your content available in a single app! Structure your feeds into categories, promote certain categories on your home screen, and build a full-featured browsing app for your users. Users will be able to comment, like and share your content to Facebook, Twitter and Foursquare with links that lead back to your website and help build traffic. Mobile App Builder offers a simple and affordable way for anyone to have a mobile app in iTunes AppStore and Android Marketplace in no time, so check it out!

Mobile App API Wordpress plugin serves as a connector between your Wordpress site and the mobile application build on Mobile App Builder. It exposes a REST API that allows your users to read blog posts from your site and comment them directly from your mobile application.

== Installation ==


1. You can download and install mobile-app-api using the built in WordPress plugin installer. If you download "mobile-app-api" manually, upload it to the "/wp-content/plugins/" directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. (recommended) Go to Settings > Mobile App API in WordPress admin area and change encryption key.
